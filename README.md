# JavaScript Платформер

Простая игра-платформер, созданная на чистом JavaScript с использованием Canvas API.

## Использование нейросети

- **Нейросеть**: DeepSeek
- **Время выполнения**: ~3 часа
- **Создано с помощью нейросети**:
  - Базовая архитектура проекта
  - Классы Game, Player, Platform, Coin
  - Система коллизий и физика
  - Игровой цикл и логика уровней
  - Система очков и прогресса
  - Визуальные эффекты и анимации
  - Документация и комментарии

## Функционал

- ✅ Управление персонажем (движение, прыжки)
- ✅ Физика (гравитация, коллизии)
- ✅ Система подсчета очков
- ✅ Несколько уровней с увеличивающейся сложностью
- ✅ Сбор предметов (монеты)
- ✅ Система жизней
- ✅ Условия победы/поражения
- ✅ Интерфейс (старт, пауза, рестарт)
- ✅ Прогресс-бар сбора монет

## Управление

- **← / A** - Движение влево
- **→ / D** - Движение вправо  
- **Пробел / W / ↑** - Прыжок
- **ESC** - Пауза

## Цель игры

Соберите все монеты на уровне, избегая падения с платформ. Каждая монета дает 10 очков. При падении теряется жизнь. После сбора всех монет переходите на следующий уровень с бонусными очками!

## Запуск

1. Скачайте все файлы в одну папку
2. Откройте `index.html` в браузере
3. Нажмите "Старт" для начала игры

## Усложнение с уровнями:

- **Уровень 2+**: Появляются опасные платформы (красные с шипами)
- **Уровень 3+**: Добавляются двигающиеся платформы (синие)
- **Уровень 4+**: Появляются враги, которые ходят по платформам
- **Уровень 5+**: Вертикальные двигающиеся платформы (фиолетовые)
- **Уровень 6+**: Летающие враги
- **Уровень 7+**: Диагональные двигающиеся платформы (оранжевые)
- **Уровень 8+**: Быстрые враги

Каждый уровень добавляет новые монеты и увеличивает сложность!
